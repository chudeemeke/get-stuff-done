  <action>
Install svgexport and export all PNG sizes:

```bash
bun add -d svgexport
```

**Export icon PNGs (dark variant):**
```bash
cd assets
for size in 16 32 64 128 256 512; do
  bunx svgexport gsd-icon.svg gsd-icon-dark-${size}.png ${size}:${size}
done
```

**Export icon PNGs (light variant):**
```bash
for size in 16 32 64 128 256 512; do
  bunx svgexport gsd-icon-light.svg gsd-icon-light-${size}.png ${size}:${size}
done
```

**Export lockup PNGs:**
```bash
# Dark variant
bunx svgexport gsd-logo.svg gsd-logo-dark-256.png 256:
bunx svgexport gsd-logo.svg gsd-logo-dark-512.png 512:
bunx svgexport gsd-logo.svg gsd-logo-dark-1200.png 1200:

# Light variant
bunx svgexport gsd-logo-light.svg gsd-logo-light-256.png 256:
bunx svgexport gsd-logo-light.svg gsd-logo-light-512.png 512:
bunx svgexport gsd-logo-light.svg gsd-logo-light-1200.png 1200:
```

Note: For lockups, specify width only (e.g., `512:`) to maintain aspect ratio.

**Verify exports:**
Check file sizes - PNGs should not be 0 bytes. Small icons (16px) may be under 1KB, larger ones several KB.
  </action>
  <verify>
```bash
ls -la assets/*.png | head -20
```
All PNG files exist and have non-zero sizes.
  </verify>
  <done>
18 PNG files exported: 12 icon sizes (6 dark + 6 light) + 6 lockup sizes (3 dark + 3 light).
  </done>
