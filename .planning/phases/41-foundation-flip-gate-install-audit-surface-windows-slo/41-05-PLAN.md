<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Test code to local shell | Tests execute local git/node/bun subprocesses with temp paths. |
| Windows process creation to test runner | Slow or hung subprocesses can block CI until timeout. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-41-05-01 | Denial of Service | subprocess-heavy tests | mitigate | Central timeout helper with killSignal and explicit timeout classification. |
| T-41-05-02 | Tampering | shell command strings | mitigate | Prefer argv arrays and isolate shell usage in `runShellWithTimeout`. |
| T-41-05-03 | Repudiation | residual raw calls | mitigate | Summary lists any raw calls left with file, line, and reason. |
</threat_model>
