    **Bug fix in scripts/preview-update.js:**

    On line 303, change:
    ```javascript
    (a, b) => (severityOrder[a.severity] || 99) - (severityOrder[b.severity] || 99)
    ```
    to:
    ```javascript
    (a, b) => (severityOrder[a.severity] ?? 99) - (severityOrder[b.severity] ?? 99)
    ```

    This is a one-character fix per occurrence (2 occurrences on the same line). The `||` operator treats `0` (the value for 'elevated' severity) as falsy, causing elevated-severity findings to sort last instead of first. The `??` operator only treats `null`/`undefined` as nullish, so `0` is preserved correctly.

    **Test additions in tests/preview-update.test.js:**

    1. Add a regression test for the severity sort fix inside the `generateReport()` describe block:
    ```javascript
    test('severity sort places elevated before high', () => {
      const { generateReport } = require('../scripts/preview-update');
      const report = generateReport({
        pinnedVersion: '1.30.0',
        latestVersion: '1.31.0',
        versionChanged: true,
        scanFindings: [
          { check: 'high-check', severity: 'high', triggered: [], evidence: [] },
          { check: 'elevated-check', severity: 'elevated', triggered: ['trigger1'], evidence: [] },
        ],
        overrideImpact: [],
      });
      const elevatedIdx = report.indexOf('elevated-check');
      const highIdx = report.indexOf('high-check');
      expect(elevatedIdx).toBeGreaterThan(-1);
      expect(highIdx).toBeGreaterThan(-1);
      expect(elevatedIdx).toBeLessThan(highIdx);
    });
    ```

    2. Fix network-dependent tests. Find any test that calls `getVersionDelta()` without explicit `pinnedVersion` and `latestVersion` parameters. The function signature is `getVersionDelta(pinnedVersion, latestVersion)`. When both are provided, it skips `readPinnedVersion()` and `queryLatestVersion()` (which calls `npm view`). Ensure ALL test calls pass both parameters explicitly:
    - If a test calls `getVersionDelta()` with no args, change to `getVersionDelta('1.30.0', '1.31.0')` (or appropriate test values)
    - If a test calls `getVersionDelta('1.30.0')` with only pinnedVersion, add a latestVersion parameter

    3. For any `runPreviewScan()` test calls that might trigger network access indirectly, ensure they pass `opts.files` and `opts.diff` parameters to avoid the `buildFileList` path that reads from node_modules (which may not exist in CI).
