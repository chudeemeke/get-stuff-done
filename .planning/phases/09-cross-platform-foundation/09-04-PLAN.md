- [ ] Statusline renders with branding, model, context bar, and directory
- [ ] Unicode characters display correctly (or ASCII fallback works)
- [ ] Colors display correctly (or plain text fallback works)
- [ ] Progress bar scales correctly from 0-100%

### 4. Hook Execution
- [ ] Pre-compact hook creates CONTINUE.md
- [ ] Pre-compact hook appends to events.log
- [ ] Update check hook spawns background process
- [ ] Statusline hook reads stdin JSON correctly

### 5. Launcher
- [ ] `gsd` command starts (after global npm install)
- [ ] Startup banner displays correctly
- [ ] Arguments pass through to claude

### 6. gsd-tools.js
- [ ] `node gsd-tools.js state load` works
- [ ] `node gsd-tools.js init new-project` works (code file detection)
- [ ] Git operations work (commit, find-phase)
