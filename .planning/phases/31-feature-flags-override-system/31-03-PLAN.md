**Structure:**

```javascript
'use strict';

const fs = require('fs');
const path = require('path');
const crypto = require('crypto');

const PROJECT_ROOT = path.join(__dirname, '..');
const DEFAULT_OVERRIDES_DIR = path.join(PROJECT_ROOT, 'overrides');
const DEFAULT_UPSTREAM_DIR = path.join(PROJECT_ROOT, 'node_modules', 'get-shit-done-cc');
```

**Functions to implement:**

1. `walkDir(dir, base)` -- Copy of the walkDir pattern (standalone, not imported from compose.js). Returns forward-slash relative paths.

2. `hashFileContent(filePath)` -- Reads file, returns SHA-256 hex string.

3. `extractHashFromReason(reasonPath)` -- Reads REASON.md, extracts SHA-256 with regex `/^- SHA-256:\s*([a-f0-9]{64})\s*$/m`. Returns string or null.

4. `extractVersionFromReason(reasonPath)` -- Reads REASON.md, extracts version with regex `/^- Version:\s*(.+?)\s*$/m`. Returns string or null.

5. `checkOverrides(opts)` -- Main function:
   - Walk overridesDir, skip .gitkeep and *.REASON.md files
   - For each override file:
     a. Check companion REASON.md exists (`overridePath + '.REASON.md'`)
     b. If missing: record as `{ relPath, status: 'missing-reason', expectedReasonPath }`
     c. If exists: extract recorded hash and version from REASON.md
     d. Check upstream file exists at `path.join(upstreamDir, relPath)`
     e. If upstream missing: record as `{ relPath, status: 'orphaned' }`
     f. Compute current upstream hash
     g. Compare: if hashes match -> 'fresh', else -> 'stale' with recordedHash, currentHash, recordedVersion, currentVersion (from upstream package.json)
   - Read upstream version from package.json
   - Return `{ ok, overrides: [...], summary: { total, fresh, stale, missingReason, orphaned } }`
   - `ok` is true only when stale === 0 AND missingReason === 0

6. `formatReport(result)` -- Formats result into the actionable report string per CONTEXT.md output format. Includes:
   - Header line
   - Per-override block with indented fields (Status, Recorded hash, Current hash, Action)
   - Missing-reason includes expected path and paste-ready template
   - Summary line with counts
   - Returns the formatted string

**CLI entry point:**
```javascript
if (require.main === module) {
  const result = checkOverrides();
  process.stdout.write(formatReport(result));
  process.exit(result.ok ? 0 : 1);
}
```

**Module exports:**
```javascript
module.exports = { checkOverrides, hashFileContent, extractHashFromReason, extractVersionFromReason, formatReport };
```
