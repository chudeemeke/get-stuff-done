Create the interlocking squares icon using hand-crafted SVG:

**Geometry (from CONTEXT.md):**
- Two hollow square frames interlocking in chain-link style
- 30-degree isometric viewing angle
- Diagonal chain orientation: upper-left to lower-right
- Equal-sized squares with thin/delicate frame thickness
- Rounded edge profile (stroke-linecap: round, stroke-linejoin: round)
- Sharp corners on the square shapes
- Transparent background, 1:1 aspect ratio

**Isometric approach:**
Use SVG transform for isometric projection. For 30-degree isometric:
- skewY(-30) for horizontal axis
- skewX(30) for vertical axis
Or use pre-calculated coordinates for cleaner paths.

**Color application:**
- Cyan (#5FD7D7) square in front at interlock
- Green (#87D787) square behind at interlock
- Flat shading: front faces lighter, back faces darker within each color
- Darker shades: cyan-dark ~#2a6b6b, green-dark ~#436b43

**Interlock visibility:**
The squares must clearly show one passing through the other. Draw back square first, then front square overlapping. The interlock point should be obvious.

**Variants to create:**
1. `gsd-icon.svg` - Full color for dark backgrounds (primary)
2. `gsd-icon-light.svg` - Adjusted colors for light backgrounds (darker: #4BBDBD, #6DBD6D)
3. `gsd-icon-mono-white.svg` - Single white color (#FFFFFF)
4. `gsd-icon-mono-black.svg` - Single black color (#000000)

Use viewBox="0 0 512 512" for reasonable default size.

Create full lockup combining icon with text:

**Layout (FYNORA style from CONTEXT.md):**
- Icon on left, text on right
- Baseline aligned
- Minimal gap between icon and text (tight lockup)
- Horizontal orientation

**Text treatment:**
- Text: "Get*Stuff*Done" where * is middle dot (Unicode: U+2022 or U+00B7)
- Title Case: Get, Stuff, Done
- Bullet separators in white/neutral color
- Tight/compact letter spacing

**Font approach (from RESEARCH.md):**
Convert text to paths for consistent rendering. Create chunky pixel-style letterforms:
- Use a 5x7 or 6x8 pixel grid per character
- Each "pixel" is a small rectangle
- Authentic 8-bit aesthetic, not smoothed

Alternatively, use a system pixel font and convert to paths, but custom paths preferred for control.

**Gradient on text:**
```svg
<defs>
  <linearGradient id="textGrad" x1="0%" y1="0%" x2="100%" y2="0%">
    <stop offset="0%" stop-color="#5FD7D7"/>
    <stop offset="100%" stop-color="#87D787"/>
  </linearGradient>
</defs>
```
Apply to text elements, NOT to bullets (bullets stay white/neutral).

**Variants:**
1. `gsd-logo.svg` - Full lockup for dark backgrounds
2. `gsd-logo-light.svg` - Adjusted for light backgrounds (darker gradient stops)

Use viewBox that accommodates both icon and text. Recommend ~1200x400 or similar wide aspect ratio for lockup.
