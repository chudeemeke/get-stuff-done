2. **autocompact_threshold ghost reference** -- `hooks/gsd-statusline.js` reads `context_management.autocompact_threshold` via `getConfigValue` with default 16.5, but ConfigSchema has this field commented out with `additionalProperties: false`. Users cannot configure it. Default works, but code suggests configurability that doesn't exist.
