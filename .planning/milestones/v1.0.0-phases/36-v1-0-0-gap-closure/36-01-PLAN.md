    **Bug fix in scripts/preview-update.js:**

    On line 303, change:
    ```javascript
    (a, b) => (severityOrder[a.severity] || 99) - (severityOrder[b.severity] || 99)
    ```
    to:
    ```javascript
    (a, b) => (severityOrder[a.severity] ?? 99) - (severityOrder[b.severity] ?? 99)
    ```

    This is a one-character fix per occurrence (2 occurrences on the same line). The `||` operator treats `0` (the value for 'elevated' severity) as falsy, causing elevated-severity findings to sort last instead of first. The `??` operator only treats `null`/`undefined` as nullish, so `0` is preserved correctly.
