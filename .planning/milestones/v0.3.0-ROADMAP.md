# Milestone v0.3.0: Upstream Sync & Workflow Maturity

**Status:** SHIPPED 2026-03-08
**Phases:** 18-23
**Total Plans:** 17

## Overview

v0.3.0 integrates 185 upstream commits (v1.10.0 through v1.20.5), adopts upstream's modular gsd-tools architecture (11 CJS domain modules), and builds mature sync workflow tooling on the new foundation. The roadmap follows a strict dependency chain: execute the massive upstream sync first (Phase 18), stabilize the build pipeline and assess what upstream now provides (Phase 19), then layer sync safety features (Phase 20), intelligent monitoring (Phase 21), and advanced automation (Phase 22) on the solid modular base. Phase 23 closed all audit gaps for release readiness.

## Milestone Summary

**Key Decisions:**

- Adopt upstream modular gsd-tools architecture (11 CJS modules) over keeping monolith
- Upstream features win on architecture conflicts unless fork approach is clearly superior
- Plumbing/porcelain split: sync.cjs for data operations, upstream-sync.md for UX orchestration
- Checkpoint lifecycle: sync-checkpoint-{batchId} tags, auto-cleaned after success
- Security keyword detection takes highest priority over conventional prefix
- Prompt Integrity requires both file-path and content match to trigger elevated severity
- CONFLICT_ANALYSIS coexists with legacy CONFLICT_DETECTED (fallback)
- AI conflict resolution always active regardless of filter state
- SYNC-II-05 scope amended: 95%+ coverage applies to production code, not test helpers
- autocompact_threshold replaced with hardcoded constant (Claude Code internal)

**Issues Resolved:**

- Phase 18 documentation/verification gaps (retroactive verification in Phase 23)
- Stale dist bundles (rebuilt in Phase 23 with Phase 21-22 features)
- ConfigLoader missing gsd section defaults
- autocompact_threshold ghost reference in statusline
- REQUIREMENTS.md stale traceability (all 16 requirements updated)
- Dollar-sign CLI test failures on Windows (execFileSync array form fix)
- Lines coverage gap (scope amendment: production code only)

**Issues Deferred:**

- Multi-runtime support (6 upstream commits skipped: Codex 5, Gemini 1) -- separate milestone
- CLAUDE-06: Full agent teams orchestration (deferred per ASSESS-01)
- PLAT-07: Interactive diff viewer (deferred per ASSESS-02)
- PLAT-08: Multi-upstream support (deferred per ASSESS-02)

**Technical Debt Incurred:**

- 6 timeout test failures on Windows (pre-existing, not v0.3.0 regression)
- 11 human verification items pending across Phases 20-22 (terminal rendering, live workflow)
- Pre-existing installer.test.js timeout failure on Windows MINGW64
