**FEAT-02 tests (opt-out model):**
- filter() with empty exclude lists preserves all manifest entries
- filter() with one item excluded preserves all OTHER manifest entries unchanged

**FEAT-03 tests (runtimes ignored):**

- filter() with runtimes set to all false still passes all files through

**Warning tests:**

- Excluding "help" in workflows.exclude removes workflow help.md but NOT commands/gsd/help.md
