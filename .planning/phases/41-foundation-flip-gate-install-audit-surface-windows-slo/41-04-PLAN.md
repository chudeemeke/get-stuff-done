| Scratch install to local machine | Bench install can run package lifecycle scripts if incorrectly invoked. |
| XML parser to test timing artifact | Bun JUnit XML is parsed into trusted timing data. |

| T-41-04-02 | Elevation of Privilege | install benchmark | mitigate | Always use scratch directories and `bun install --ignore-scripts`. |
| T-41-04-03 | Information Disclosure | benchmark logs | accept | Commands expose timings and versions only; no secrets are read or printed. |
