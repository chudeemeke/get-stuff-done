## Decisions Made

- Uninstall removes all entries from target but preserves the empty target directory itself (consistent with upstream behavior)
- computeDelta() tracks .overlay-manifest.json as always-modified alongside .install-meta.json (both regenerate on every compose)
- CREDITS.md included in overlay manifest as an overlay-only additive file
