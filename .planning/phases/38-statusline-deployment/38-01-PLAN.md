Move fork-specific hooks into the overlay directory, fix timeouts, and update all downstream tooling references so nothing breaks after the move.

Purpose: The fork's enhanced statusline (254 lines) and check-update hook (193 lines with maintainer git fetch) currently live at `hooks/` in the repo root, outside the `overlay/` directory the composition pipeline walks.
