**Check 1: checkPromptIntegrity(diff, files)**
- PROMPT_DIRS: `['workflows/', 'agents/', 'commands/', 'templates/']`
- File-path gate: at least one file must be a `.md` file in a PROMPT_DIR. If no file matches, return null immediately.
- Content checks on diff text (any ONE match triggers elevated finding):
  - `injection-pattern`: `/ignore previous instructions?/i`, `/override (your|all) instructions?/i`, `/new instructions?:/i`, `/you are now/i`, `/disregard/i`
  - `hidden-unicode`: `/[\u200b-\u200f\u202a-\u202e\u2060-\u2064\ufeff]/`
  - `credential-expand`: `/~\/\.ssh|~\/\.aws|\.env|credentials/`
  - `tool-list-change`: `/^[+-].*tools:/m`
  - `guardrail-removal`: `/^-.*\b(safe|blocked?|forbidden|disallow|prevent|restrict)\b/im`
- File-path match without any content check match = return null (not elevated, just informational -- per Pitfall 5 in RESEARCH.md)
- Return: `{ triggered: string[], evidence: string[] }` where evidence = specific lines from diff that matched

Tests:
- Diff modifying `workflows/foo.md` with "ignore previous instructions" => triggers with injection-pattern
- Diff modifying `agents/bar.md` with zero-width chars => triggers with hidden-unicode
- Diff modifying `templates/x.md` with `+tools: Read, Write, Bash, Execute` => triggers with tool-list-change
- Diff modifying `workflows/y.md` with `-  blocked: true` => triggers with guardrail-removal
- Diff modifying `workflows/z.md` with only normal text changes => returns null (no content match)
- Diff modifying `src/foo.js` with "ignore previous instructions" => returns null (not in PROMPT_DIRS)
- Diff modifying `docs/readme.md` with injection patterns => returns null (docs/ is NOT in PROMPT_DIRS)

**Check 2: checkDependencyDiff(diff, files)**

**Implement runSupplyChainChecks orchestrator:**

```javascript
function runSupplyChainChecks(diff, files, authorString, knownAuthors) {
  const findings = [];

  // Diff size guard (Pitfall 3 from RESEARCH.md):
  // If diff > 500KB, skip content-based pattern matching but still run file-path checks
  const diffTooLarge = diff.length > 500 * 1024;
  const safeDiff = diffTooLarge ? '' : diff;

  // Check 1 (ELEVATED -- primary check, highest display prominence)
  const promptRisk = checkPromptIntegrity(safeDiff, files);
  if (promptRisk) findings.push({ check: 'prompt-integrity', severity: 'elevated', ...promptRisk });

  // Check 2
  const depRisk = checkDependencyDiff(safeDiff, files);
  if (depRisk) findings.push({ check: 'dependency-diff', severity: 'high', ...depRisk });

  // Check 3 (file-path only -- runs even on large diffs)
  const execRisk = checkExecutionPath(files);
  if (execRisk) findings.push({ check: 'execution-path', severity: 'high', ...execRisk });

  // Check 4
  const netRisk = checkNetworkEndpoints(safeDiff);
  if (netRisk) findings.push({ check: 'network-endpoints', severity: 'medium', ...netRisk });

  // Check 5
  const obfRisk = checkObfuscation(safeDiff, files);
  if (obfRisk) findings.push({ check: 'obfuscation', severity: 'high', ...obfRisk });

  // Check 6
  const authorResult = checkAuthorAnomaly(authorString, knownAuthors);
  if (!authorResult.isKnown) {
    findings.push({ check: 'author-anomaly', severity: 'medium', author: authorResult.author });
  }

  // If diff was too large, add an informational finding
  if (diffTooLarge) {
    findings.push({ check: 'diff-size', severity: 'info', triggered: ['diff-too-large'], evidence: [`Diff is ${Math.round(diff.length / 1024)}KB, content checks skipped`] });
  }

  return findings;
}
```

Note: The orchestrator receives author + knownAuthors as parameters. The cmdSyncPreview caller is responsible for loading the author cache and passing it. This keeps runSupplyChainChecks pure and testable.
