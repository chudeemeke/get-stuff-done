## Implementation

- Added `scripts/lib/upstream-source.js` as the maintainer/build-time authority helper.
- Helper reads `.planning/upstream-authority.json`, validates exact stable semver,
  rejects mutable tags/prerelease pins, rejects absolute/traversal paths, resolves
  scoped package paths, exposes bin/path mappings, and derives compose category
  directories from the authority contract.
- Added `active.paths.scripts = "scripts"` to the authority contract/schema so
  required Open GSD top-level entries are not hidden in script-local constants.
- Replaced `get-shit-done-cc@1.39.1` with exact-pinned `@opengsd/gsd-core@1.5.0`.
- Migrated compose/check-overrides/check-boundary defaults to the helper-backed
  package directory.
- Migrated `preview-update.js` to query npm with:

```javascript
execFileSync('npm', ['view', ACTIVE_UPSTREAM_PACKAGE, 'version'], ...)
```

  This avoids shell parsing of scoped package names.
- Updated active branding rules from legacy package/repo substitutions to:
  `@opengsd/gsd-core -> @chude/get-stuff-done` and
  `open-gsd/gsd-core -> chudeemeke/get-stuff-done`.
- Refreshed hook override snapshot hashes against `@opengsd/gsd-core@1.5.0`.
