<task type="auto">
  <name>Task 2: GREEN -- Implement v3.0 delegation installer</name>
  <files>bin/install.js</files>
  <action>
Rewrite bin/install.js as the v3.0 delegation-based installer (~200-300 lines). The v2.x content (2,125 lines) is completely replaced.

Implementation structure:

**Argument parsing:**
- Parse process.argv for: --claude, --opencode, --gemini, --all, --global, --local, --config-dir, --uninstall, --force (non-interactive mode for testing/scripting)
- All flags pass through to upstream unchanged

**resolveTargetDir(args):**
- Replicate upstream's getGlobalDir() resolution priority:
  1. --config-dir flag value (if provided)
  2. CLAUDE_CONFIG_DIR env var
  3. os.homedir() + '/.claude' (default for --claude --global)
- For --local: process.cwd() + '/.claude' (or runtime-specific dirname)
- Return the resolved target directory path

**detectV2(targetDir):**
- Signal 1 (primary): Check targetDir/get-stuff-done/.install-meta.json -- if exists and (missing overlay_version OR version < 3.0), return { isV2: true, signal: 'meta', version }
- Signal 2 (fallback): Check for src/ directory in targetDir (v2.x installed fork source)
- Signal 3 (fallback): Check for get-stuff-done/ dir without get-shit-done/ dir (v2.x directory name)
- Check all three .install-meta.json paths per research Pitfall 4: target/get-stuff-done/.install-meta.json (v2.x), target/.install-meta.json (v3.0 composition), target/get-shit-done/.install-meta.json (upstream-only)
- Return { isV2: false } if none match

**cleanupV2(targetDir):**
- Prompt user with what will be removed (per CONTEXT.md: "require confirmation before proceeding")
- Skip prompt if --force flag is set (non-interactive)
- If confirmed: fs.rmSync(targetDir, { recursive: true, force: true }) -- wipe entire install dir
- User config (~/.gsd/) and project data (.planning/) are outside blast radius

**install(distDir, targetDir, args):**
- Read .overlay-manifest.json from distDir to get overlay-only file list
- Spawn `node distDir/bin/install.js` with all user args + stdio: 'inherit'
- On 'close' with code 0:
  - Copy each overlay-only file from distDir to targetDir, creating subdirectories as needed
  - Write .install-meta.json to targetDir/get-shit-done/ with: upstream_version (from dist/.install-meta.json), overlay_version (from own package.json), installed_at (ISO timestamp), features_disabled (from dist/.install-meta.json), overrides_applied (from dist/.install-meta.json)
- On 'close' with non-zero code: warn about possible partial state, exit with upstream's exit code

**uninstall(targetDir):**
- If targetDir doesn't exist: log "Nothing to uninstall", exit 0
- fs.rmSync(targetDir, { recursive: true, force: true })
- No confirmation prompt per CONTEXT.md (--uninstall flag is sufficient intent)
- Exit 0

**Main flow:**
1. Determine distDir: path.join(__dirname, '..', 'dist')
2. Verify distDir exists (error with guidance to run `bun run compose` first)
3. Parse args, resolve target dir
4. If --uninstall: call uninstall(), exit
5. If target exists: check for v2.x, clean if needed
6. Call install()

Use child_process.spawn (NOT execSync) for the upstream subprocess per CONTEXT.md locked decision. stdio: 'inherit' for real-time output passthrough.
